This project demonstrates a RESTful API for managing user data in the Chandigarh University database system. It supports full CRUD operations using standard HTTP methods.

The API allows you to:

  Retrieve user data
  Insert new records
  Update existing records
  Delete records
  
Technologies Used:---
    Backend: SpringBoot/Java
    Database: MySQL
    API Testing: Postman
    Command Line: Windows CMD
